# 교육청 법카맵 v3.4

서울교육 업무추진비 공개자료를 가져와 사용장소 지도와 결과 목록으로 시각화하는 실험형 도구입니다.

## v3.4 변경사항

- 자동 수집 중 `자료 수집 중...` 점 애니메이션 표시
- 자동 수집이 오래 걸리면 브라우저에서 중단/실패 처리
- `수집 중지`, `다시 시도`, `새로 시작하기` 흐름 추가
- 자동 수집 성공 후 `이 자료로 지도 만들기` 버튼을 바로 노출
- 지도 생성 후 결과 탭 기본값을 `지도표시`로 변경
- 탭 순서를 `지도표시 · 확인필요 · 제외 · 전체`로 정리
- 지도 팝업을 X 버튼으로 닫을 수 있게 변경
- 지도 빈 곳 클릭 시 팝업 닫기, 마커 클릭 시 팝업 1개만 표시
- 결과 카드 클릭 시 지도 마커 위치로 이동
- v3.3의 Kakao 키 자동저장, 전체 시트 합쳐 읽기 유지

## 실행

처음 1회:

```powershell
py -m pip install -r requirements.txt
py -m playwright install chromium
```

실행:

```powershell
py server.py
```

브라우저:

```text
http://localhost:3000
```

## Kakao Developers 도메인

로컬 테스트:

```text
http://localhost:3000
http://127.0.0.1:3000
```

GitHub Pages/Vercel 배포 시 실제 배포 도메인도 추가해야 합니다.
