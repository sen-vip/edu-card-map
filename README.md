# 교육청 법카맵 v3.8.3

## v3.8.3 모바일 최적화
- 모바일 상단에 자료·지도·표 빠른 이동 바를 추가했습니다.
- 모바일에서 히어로와 카드 여백을 줄여 첫 화면에서 자료 가져오기 흐름이 더 빨리 보이게 했습니다.
- 진행 단계는 세로로 길게 내려가지 않고 가로 스크롤 칩 형태로 보이게 바꿨습니다.
- 지도 영역을 모바일 화면 높이에 맞춰 크게 보이게 조정하고, 선택 장소 패널은 지도 아래 보조 카드로 정리했습니다.
- 통계 카드와 결과 요약 카드는 가로 스크롤 칩으로 바꿔 화면을 덜 차지하게 했습니다.
- 결과 표는 모바일에서 카드형 행으로 접혀 보이도록 하여 가로 스크롤 없이 상호·일시·금액·주소·작업을 확인할 수 있게 했습니다.
- 하위항목 콤보박스는 모바일에서 화면형 목록으로 열려 직속기관·도서관까지 찾기 쉽게 했습니다.


## v3.8.2
- 결과 요약 카드도 클릭할 수 있게 바꿔 전체/지도표시/확인필요/제외 필터로 바로 이동합니다.
- 표 행을 누르면 지도 영역으로 이동하고, 해당 장소 말풍선과 선택 장소 카드가 함께 열리도록 했습니다.
- 지도 마커 클릭은 기존처럼 아래 목록으로 이동하지 않고 지도 안에서만 확인됩니다.
- 상태 배지의 글자가 두 줄로 떨어지지 않도록 표 첫 열 폭과 배지 줄바꿈을 보정했습니다.
- 하위항목 콤보박스는 교육지원청·직속기관·도서관이 먼저 보이도록 순서를 조정하고, 전체 목록 높이를 늘렸습니다.

## 이번 버전
- 버튼명을 “첨부 엑셀 자동 가져오기”에서 “공개자료 가져오기”로 변경했습니다.
- 하위항목 입력칸 폭을 줄여 조건 행이 과하게 길어지지 않도록 조정했습니다.
- 공개자료 가져오기 버튼을 하위항목 오른쪽에 배치해 조건 선택 후 바로 실행하는 흐름으로 바꿨습니다.
- 보조 버튼은 다음 줄로 낮춰 기본 동선의 시선을 분산하지 않게 정리했습니다.
- 교육감/부교육감처럼 하위항목이 없는 게시판에서는 기준월·게시판·자동 가져오기만 한 줄로 보이게 정리했습니다.

## v3.7.9
- 공개자료를 자동으로 가져오거나 엑셀을 직접 불러오면 별도 버튼 없이 바로 지도 생성까지 진행합니다.
- 지도 만들기 버튼은 보조용으로 오른쪽에 유지하고, 지도 생성 후에는 “지도 다시 만들기”로 표시합니다.
- 결과 영역 기본 보기를 카드가 아닌 표로 고정했습니다.
- 카드/표 전환 UI를 제거해 지도 확인 + 표 검토 흐름을 단순화했습니다.

# 교육청 법카맵

서울교육 업무추진비 공개자료를 자동으로 가져와 지도와 목록으로 확인하는 웹앱입니다.

## v3.7.7 결과 UI 개선

- 추출 결과 상단에 전체/지도표시/확인필요/제외/금액 요약 카드를 추가했습니다.
- 상태 필터에 건수를 표시해 현재 어떤 결과를 보고 있는지 바로 알 수 있게 했습니다.
- 결과 목록을 문단형 텍스트에서 카드형 검토 UI로 바꿨습니다.
- 카드/표 보기 전환과 최신순·금액 큰 순·상호명순 정렬을 추가했습니다.
- 카드별로 지도에서 보기, 주소 복사, 확인필요, 제외 액션을 추가했습니다.
- 표 보기에는 고정 헤더, 상태 배지, 축약 표시를 적용해 많은 결과를 비교하기 쉽게 정리했습니다.

## v3.7.0 Vercel 백엔드 준비판

이번 버전은 GitHub Pages 정적 배포에서 빠졌던 `/api` 백엔드를 Vercel 배포 구조로 보완했습니다.

### 반영 내용

- `/api/sen-browser` 추가: 프론트의 자동 수집 버튼과 연결
- `/api/sen-find` 추가: 게시글 후보 찾기 버튼과 연결
- `/api/sen-auto` 정리: 직접 URL 불러오기와 공통 자동 다운로드 담당
- `lib/sen-scraper.js` 재사용: 열린교육 게시판 검색, 상세글 확인, 첨부파일 다운로드
- `vercel.json` 추가: Vercel Functions 실행 시간과 API 캐시 방지 설정
- `server.js` 추가: 로컬에서도 Vercel API와 비슷한 방식으로 테스트 가능
- `README.md`를 Vercel 배포 기준으로 정리

## Vercel 배포

1. GitHub 저장소에 이 파일들을 업로드합니다.
2. Vercel에서 `Add New Project`를 누릅니다.
3. `edu-card-map` 저장소를 선택합니다.
4. Framework Preset은 `Other`로 두고 배포합니다.
5. 배포 주소가 나오면 Kakao Developers의 JavaScript 키 허용 도메인에 추가합니다.

예시:

```txt
https://edu-card-map.vercel.app
```

## 로컬 테스트

Node.js 18 이상에서 실행합니다.

```bash
npm install
npm run dev
```

브라우저에서 접속합니다.

```txt
http://localhost:3000
```

## Kakao 지도 키

Kakao JavaScript 키는 브라우저 `localStorage`에 저장됩니다. Vercel 배포 주소를 Kakao Developers의 허용 도메인에 등록해야 지도 표시가 정상 작동합니다.

## 주의

- 열린교육 게시판 구조가 바뀌면 자동 수집이 실패할 수 있습니다.
- GitHub Pages 주소에서는 자동 수집 API가 실행되지 않습니다. Vercel 배포 주소를 사용하세요.
- 자동 수집이 막히면 엑셀을 직접 내려받아 업로드할 수 있습니다.
- 위치는 Kakao 장소 검색 결과를 바탕으로 추정하므로 확인필요 항목은 원자료와 대조하세요.


## 3.7.3 수정

- 게시글 후보를 찾았지만 상세 URL이 HTML에 직접 노출되지 않는 경우, 즉시 실패하지 않고 Vercel 브라우저 방식으로 실제 제목 클릭과 첨부 다운로드를 재시도합니다.
- 열린서울교육 게시판처럼 제목 링크가 일반 href로 노출되지 않는 CMS 구조를 보완했습니다.


## v3.7.6 업데이트

- 열린교육 게시글 제목 클릭 시 발생하는 네트워크 요청/응답을 기록합니다.
- URL이 `view0010v.do`까지만 보이고 게시글 번호가 숨겨지는 경우를 대비해, POST payload와 응답 HTML을 검사합니다.
- 실제 상세 HTML에 선택한 제목이 포함되어 있는지 검증한 뒤 첨부파일 탐지를 시작합니다.
- 실패 시 `목록 행 확인`, `클릭 대상 속성`, `네트워크 중요 기록`, `상세 HTML 후보` 로그를 남겨 원인 진단이 가능하도록 했습니다.
